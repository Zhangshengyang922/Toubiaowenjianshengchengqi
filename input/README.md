# 招标文件投放区

把你收到的招标文件（PDF / DOCX / TXT / XLSX）拖入这个文件夹，
然后运行：

    python process_all.py

程序会自动：
1. 扫描并识别哪些是招标文件
2. 解析其中的内容要求和格式要求
3. 按招标文件的要求生成对应的投标文件 (DOCX)
4. 输出到 ../output/ 文件夹

---

## 支持的招标文件格式

| 格式 | 说明 |
|------|------|
| .pdf | PDF格式招标文件 |
| .docx | Word格式招标文件 |
| .txt | 纯文本招标文件 |
| .xlsx | Excel格式（招标清单等） |

## 使用技巧

- 文件名建议包含项目名称，方便识别
- 可以将多个招标文件一起放入，批量处理
- 先用 --scan-only 预览识别结果，确认无误后再生成：
  ```
  python process_all.py --scan-only
  ```
